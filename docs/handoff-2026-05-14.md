## Gotchas / cosas que rompen el sistema

1. **Suspensión del PC durante batch Ollama** → Ollama se cuelga, hay que `sudo systemctl restart ollama`. Mitigación: `setsid nohup ... < /dev/null & disown` pero igual cuelga si suspends.

2. **Postgres container parado** → eval falla con "connection refused". Levantar con `docker start energy_rag_pg`.

3. **GPU full** → qwen3.5:9b ocupa 6.4/8GB. Si cargas otro modelo simultáneo (ej. Qwen3-Embedding), GPU se llena. Por eso embedder está forzado a CPU en `.env`.

4. **Eval Ollama no-determinístico** ±3pp entre runs. NO comparar una sola eval contra otra como medición precisa.

5. **Push falla por auth** → renueva token GitHub. Commits locales sobreviven.
